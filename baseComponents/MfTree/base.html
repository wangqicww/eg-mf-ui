# slot

  • 武汉市
    • 洪山区
    • 高新区
    • 江夏区
    • 汉阳区
<template>
  <div class="egova-mf-title-demo" style="min-height:30px;width:100%;padding:10px;">
      <mf-tree
          ref="test"
          :tree-data="treeData"
          :load-data="onLoadData"
          checkable
          iconCls="mf-icon-urge"
          @select="select"
      >
          <template slot-scope="{ data }">
              <div class="mf-tree__btn" @click="deleteNode(data)">删除</div>
              <div class="mf-tree__btn">编辑</div>
          </template>
      </mf-tree>
  </div>
</template>
<script>
export default {
    data() {
        return {
            defaultCheckedKeys: [3,4],
            treeData: [
                {
                    id: 1,
                    text: "武汉市",
                    children: [
                        {id: 2, text: "洪山区", iconCls: "icon-leaf", canEdit: true, isLeaf: true},
                        {id: 3, text: "高新区", canEdit: true},
                        {id: 4, text: "江夏区"},
                        {id: 5, text: "汉阳区"},
                    ]
                }
            ]
        }
    },
    methods: {
        onLoadData(treeNode) {
            return new Promise(resolve => {
                const data = treeNode.dataRef;
                if (data.iconCls === "icon-leaf") {
                    resolve();
                    return;
                }
                data.children = [
                    { id: `${treeNode.eventKey}-0`, text: `${data.text}街道1`, iconCls: "icon-leaf" },
                    { id: `${treeNode.eventKey}-1`, text: `${data.text}街道2`, iconCls: "icon-leaf" },
                ];
                resolve();
            });
        },
        select() {},
        deleteNode() {}
    }
}
</script>

# props

属性 说明 类型 传值 默认值
treeData 数据 Array 非必传 -
defaultExpandedKeys 默认展开节点 Array 非必传 -
defaultCheckedKeys 默认选中节点 Array 非必传 -
iconCls 图标类名 String 非必传 mf-icon-unit
replaceFields 替换 treeNode 中 title,key,children 字段为 treeData 中对应的字段 Object 非必传 {children:'children', title:'title', key:'key' }

# treeData props

属性 说明 类型 传值 默认值
canEdit 是否可操作，显示节点末尾（...） Boolean 非必传 -
iconClass 节点图标，优先读取此字段，没有读取iconCls属性 String 非必传 -

# treeData说明

说明
有必要的话，自行对数据进行格式化，如果节点没有scopedSlots， 那么节点默认对数据做处理：data.scopedSlots = { title: "custom", icon: "icon" }

# event

名称 说明 回调参数
getCheckedKeys 获取选中节点key [keys]

# slot

名称 说明
default 树操作蓝内容

# slot说明

说明
操作按钮增加class=’mf-tree__btn‘， 此class有默认样式
