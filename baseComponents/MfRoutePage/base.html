# props

属性 说明 类型 传值 默认值
route-search-config 查询栏配置项 Object 非必传 -
route-operate-config 操作栏配置项 Object 非必传 -
route-table-config 表格配置项 Object 非必传 -
route-form-config 表单配置项 Object 非必传 -
route-page-config 详情配置项 Object 非必传 -
service-config 请求配置项 Object 非必传 -

# route-search-config props

属性 说明 类型 传值 默认值
search 具体配置见search组件，route-page有默认的处理search事件，外部传入可覆盖内置方法 Object 非必传 -

# route-operate-config props

属性 说明 类型 传值 默认值
route-operate-config 具体见operate组件，默认layout: "add, batchDel, exp"， 并有相应的处置事件，外部传入可覆盖内置方法 Object 非必传 layout: "add, batchDel, exp"

# route-table-config props

属性 说明 类型 传值 默认值
route-operate-config 具体见layout组件，默认表格已处理，外部传入会覆盖 Object 非必传 layout: "add, batchDel, exp"

# route-form-config props

属性 说明 类型 传值 默认值
uploadConfig 具体见form组件, 内置了删除附件方法 Object 非必传 uploadConfig: {label: "附件", deleteMedia: this.deleteMedia }

# service-config props

属性 说明 类型 传值 默认值
serviceBase 请求axios对象 Object 非必传 -
queryMethodName 查询方法名称 String 非必传 -
delMethodName 删除方法名称 String 非必传 -
addMethodName 新增方法名称 String 非必传 -
editMethodName 编辑方法名称 String 非必传 -
formatTableQueryParams 数据查询前的格式化函数钩子函数，返回false终止查询 Function 非必传 search组件数据
formatDelParams 删除数据前的格式化函数钩子函数，返回false终止删除 Function 非必传 当前要删除数据
formatFormData 删除新增编辑前的格式化函数钩子函数，返回false终止新增编辑 Function 非必传 新增和编辑数据
exportMethodName 导出方法名称 String 非必传 -
exportFileName 导出文件名称 String 非必传 文件
mediaServiceBase 附件请求axios对象 String 非必传 -
mediaUploadMethodName 附件上传方法名称 String 非必传 uploadMedia
mediaGetMethodName 附件查询方法名称 String 非必传 getMedia
mediaDelMethodName 附件删除方法名称 String 非必传 delMedia
mediaDefaultURL 附件地址通过mediaDefaultURL+mediaUrl拼接而成，不传默认使用多媒体的mediaDefaultURL字段，推荐传入mediaDefaultURL参数 String 非必传 -
uploadData 附件上传参数 String Object -
formatUploadData 附件上传前的格式化函数钩子函数，返回false终止 Function 非必传 （新增编辑返回的数据，uploadData)
formatGetMediaParams 附件获取前的格式化函数钩子函数，返回false终止 Function 非必传 当前记录
formatDelMediaParams 附件删除前的格式化函数钩子函数，返回false终止 Function 非必传 附件
formatTableData 表格数据格式函数，默认读取res.data.resultList Function 非必传 接口返回接结果
formatTotal 总数格式函数，默认读取res.data.pageInfo.totalRecord Function 非必传 接口返回接结果
tableDataLoadingComplete 表格数据加载完回调函数 Function 非必传 tableConfig

# event

名称 说明 回调参数
routeRecordDetail 查看详情、编辑、新增触发 查看详情、编辑返回当前数据，新增返回{}

# slot

名称 说明
header 详情页右上角内容
content 详情页中部内容
detailRightContent 详情页右侧内容
