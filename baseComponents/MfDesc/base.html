# mf-desc props

属性 说明 类型 传值 默认值
title 标题名称 String 非必传 -
column 一行 Descriptions Item 的数量，只支持1、2、3、4 Number 非必传 3
labelWidth 标签文本宽度，优先级高于desc-item String 非必传 -
labelPosition 标签文本对齐方式，优先级高于desc-item String 非必传 left

# mf-desc slot

名称 说明
title 自定义描述信息，不传就不显示标题

# mf-desc-item props

属性 说明 类型 传值 默认值
showColon 是否显示冒号 Boolean 非必传 true
label 标签文本 String 非必传 -
labelWidth 标签文本宽度 String 非必传 -
labelPosition 标签文本对齐方式 String 非必传 left
isExclusiveLine 是否独占一行 Boolean 非必传 false

# mf-desc-item slot

名称 说明
extra 自定义属性信息
