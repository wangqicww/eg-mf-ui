# mf-desc props

属性 说明 类型 传值 默认值
searchList 搜索列表 Array 非必传 -
showBtn 显示按钮 Boolean 非必传 true
defaultData 默认值 Object 非必传 -

# searchList props

属性 说明 类型 传值 默认值
label 字段显示名称 String 必传 -
field 字段属性 String 必传 -
type 表单类型 String，可选值有 select、datePicker、yearPicker、monthPicker、rangePicker、treeNode、treeCheck 非必传 input
options 下拉选项，type为select有效 Array type为select必传 -
mode 下拉列表模式 default、multiple、tags、combobox 非必传 default
optionId 指定options的key和value String 非必传 id
optionName 指定options的显示名称 String 非必传 name
treeData type为treeNode、treeCheck、treeAsync的选择树 String 非必传 -
fieldRelateName 下拉显示名称数据 String 非必传 -
allowClear 显示清空图标 Boolean 非必传 true
showSearch 树选择显示查询框 Boolean 非必传 false
years 年份选择列表，如[2023,2024,2025] Array 非必传 -
treeDefaultExpandedKeys treeNode和treeCheck树默认展开节点 Array 非必传 -
