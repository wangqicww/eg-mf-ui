# props

属性 说明 类型 传值 默认值
defaultFiles 默认文件类型， 参考attach，需要增加参数mediaID 非必传 Array []
acceptFile 支持文件格式，如 png,pdf,xlsx 非必传 String -
maxFileSize 支持最大文件大小（M） 非必传 Number 5
