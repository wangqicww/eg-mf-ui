# slot

prefix

<template>
    <div class="egova-mf-operate-demo" style="min-height:40px;width:100%;padding:10px;">
        <mf-operate layout="del, add, edit, exp" @add="add" :suffix-btn="prefixBtn" prefixClass="prefix-class">
            <template v-slot:prefix>
                <div>prefix</div>
            </template>
        </mf-operate>
    </div>
</template>

<script>
export default {
    data () {
        return {
            prefixBtn: [
                {name: "后置新增", icon: "mf-icon-add", color: "#AA8DFF", click: this.add1}
            ]
        }
    },
    methods: {
        add() {},
        add1() {}
    }
}
</script>
<style lang="scss">
.egova-mf-operate-demo {
    .prefix-class {
        color: red;
        font-weight: 600;
        font-size: 20px;
    }
}
</style>

# props

属性 说明 类型 传值 默认值
layout 按钮, 支持 add、del、edit、exp、batchDel、imp、downTemp String 非必传 -
prefixBtn 前置按钮，格式 [{name: "新增1", icon: "mf-icon-add", color: "#AA8DFF", click: this.add1}] Array 非必传 -
suffixBtn 后置按钮，格式同上 Array 非必传 -
prefixClass 前缀样式，结合prefix插槽使用 String 非必传 egova-mf-operate__prefix
suffixClass 后缀样式，结合suffix插槽使用 String 非必传 egova-mf-operate__suffix

# event

名称 说明 回调参数
add 新增按钮回调函数 -
edit 编辑按钮回调函数 -
del 删除按钮回调函数 -
exp 导出按钮回调函数 -
batchDel 批量删除按钮回调函数 -
imp 导入按钮回调函数 -
downTemp 下载模板按钮回调函数 -

# slot

名称 说明
prefix 前置内容
suffix 后置内容
