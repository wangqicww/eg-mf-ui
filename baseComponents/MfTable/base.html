# props

属性 说明 类型 传值 默认值
tableConfig 默认值格式如下格式，请参考a-table传入参数 Object 必传 -
columns: [],
rowKey: constant.ROW_KEY,
pagination: {
    current: 1,
    pageSize: 20,
    showQuickJumper: true,
    showSizeChanger: true,
    defaultCurrent: 1,
    defaultPageSize: 20,
    pageSizeOptions: ["20", "50", "80", "100"],
    total: 0,
    showTotal: (total) => `${total}`
}

# column props

属性 说明 类型 传值 默认值
key 字段属性 必传 String -
title 字段名称 必传 String -
check 能否反查（文字变蓝可点击） 非必传 Boolean false
checkData 反查方法 非必传 Function -
format 格式化方法, 不推荐，推荐通过遍历数据格式化 非必传 Function -
cellPrefixClass 数据前面div样式，当前与格式化方法不能共存 非必传 String -
cellClass 单元格类名 非必传 String -
cellStyle 单元格样式 非必传 String -
width 单元格宽度 非必传 String -

# slot

名称 说明
default 操作栏的默认插槽，如果operateBtn包含click属性，且是函数，点击事件优先触发此方法，并阻断btnClick方法
