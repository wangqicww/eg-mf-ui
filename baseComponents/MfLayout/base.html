# props

属性 说明 类型 传值 默认值
searchConfig 查询栏配置项 Object 非必传 -
operateConfig 操作栏配置项 Object 非必传 -
mfTableConfig 表格配置项 Object 非必传 -

# searchConfig props

属性 说明 类型 传值 默认值
searchList 具体查看search组件 - - -
showBtn 具体查看search组件 - - -
search 具体查看search组件 - - -
hideSearch 是否隐藏查询栏 boolean - false

# operateConfig props

属性 说明 类型 传值 默认值
layout 具体查看layout组件，属性都支持 - - -
hideOperate 是否隐藏操作栏 boolean - false

# mfTableConfig props

属性 说明 类型 传值 默认值
tableConfig 具体查看table组件，属性都支持 - - -
hideTable 是否隐藏表格 boolean - false

# slot

说明
search、operate、table的具名和作用域插槽都支持
